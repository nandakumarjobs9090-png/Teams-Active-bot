# Mouse Activity Keeper (Python)

A lightweight Python utility that prevents system idle by periodically moving the mouse.

## Features
- Start / Stop controls
- Configurable timer
- Runs hidden in background
- No external libraries required
- No admin rights needed

## Requirements
- Python 3.x (Windows)
